Atacuri la nivel de rețea

Există două mari categorii de atacuri la nivel de rețea:

  • DoS = Denial–of–Service
  • DDoS = Distributed–Denial–of–Service

Principiile din spatele celor două tipuri de atacuri este similar. DoS / DDoS pot fi definite ca atacuri online prin care se încearcă deturnarea unei resurse sau a unei rețele astfel încât utilizatorii să nu mai aibă acces. Acest lucru se face prin perturbarea temporară sau permanentă a serviciului unui host conectat la internet. Astfel, se încercă „inundarea” rețelei cu cereri de accesare superflue într-o pentru suprasolicitarea sistemului.

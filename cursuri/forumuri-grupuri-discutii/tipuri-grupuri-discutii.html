Tipuri de grupuri de discuții

Aceasta nu este o clasificare exhaustivă, dar acoperă o parte dintre tipurile de grupuri de discuții care există acum pe Web.

  • Forumul/message board-ul: este un site dedicat discuțiilor, de cele mai multe ori pe un anumit subiect. Pot să fie specializate/de nișă (ex.: forumul pasionaților de pus murături) sau pe teme generale. Există mai multe feluri de utilizatori, cu diferite privilegii.
  • Image board: un similar cu un message board, dar pentru a deschide o discuție este obligatorie postarea unei imagini.
  • Chat room: o formă de comunicare sincronă. Scopul unui chat room este cel de diseminare de informație, de cele mai multe ori în format text, în interiorul unui grup de utilizatori.

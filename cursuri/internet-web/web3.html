Web 3.0

Web 3.0, sau Web-ul executabil, este caracterizat de o schimbare în modul de interacțiune, de la utilizator-mașină la mașină-mașină. În acest context, computerele pot interpreta informațiile, fiind dat un volumul mare de date.

Există câteva concepte care sunt foarte des asociate Web 3.0

  • Web-ul semantic: tehnologii Web prin care informațiile pot fi citite și interpretate de computere, prin intermediul metadatelor.
  • Inteligență artificială: termen folosit adesea să descrie computere care mimează funcții cognitive specifice oamenilor, lucru posibil prin „învățare” (machine learning).

Web-ul

Atunci când folosim cuvântul internet și nu ne referim la rețeaua de rețele ar trebui să folosim, de fapt, termenul World Wide Web (www), sau mai simplu, Web. Web-ul este un mod de a accesa informații și conținut folosind internetul. Astfel, el este un model de diseminare a informațiilor construit folosind rețeaua numită internet. Web-ul folosește protocolul HTTP (Hypertext Transfer Protocol), scrisă în limbajul standardizat HTML (HyperText Markup Language). Astfel, HTML este limba Web-ului, iar HTTP este gramatica limbii.

Ce este HTTPS? Ce înseamnă? La ce ajută?

Gândiți-vă la Web ca la o colecție de documente (pagini Web) legate între ele prin hyperlink-uri. Informația se află pe niște calculatoare numite servere.

Termenii internet și Web nu ar trebui folosiți interșanjabil pentru că descriu tehnologii diferite: internetul este rețeaua de rețele, iar Web-ul este in sistem standardizat de accesare și navigare a internetului.

Web-ul nu este singurul astfel de sistem care folosește internetul. De exemplu, serviciile de email sau aplicațiile mobile nu accesează Web-ul. Mai clar, atunci când sunt folosite link-uri și vedem utilizarea protocolului HTTP atunci este vorba despre Web.

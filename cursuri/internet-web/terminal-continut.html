De la terminal de la conținut

Schema de mai jos ne arată traseul de la un terminal la conținut.

Rețeaua este infrastructura necesară pentru a putea comunica folosind un terminal, adică partea hardware.

Accesul la internet este dependent de existența unei rețele funcționale și de acces la rețea din partea unui provider de internet.

Web-ul cuprinde toate informațiile care pot fi accesate prin intermediul internetului.

Un browser este software-ul prin care este accesat Web-ul.

Un motor de căutare este un serviciu prin care se pot decoperi pagini Web fără a știi adresa exactă, prin intermediul cuvintelor cheie.

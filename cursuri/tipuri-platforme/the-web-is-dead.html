The Web is dead

În 2011, autorii Chris Anderson și Michale Wolff publicau în revista WIRED articolul The Web is dead! Long live the internet!.

Internetul este:

  • o rețea globală de rețele și computere
  • infrastructura hardware a rețelei
  • informația este transmisă prin protocoale specifice rețelei

Web-ul este:

  • o colecție de informații care pot fi accesate prin intermediul internetului
  • este nevoie de un software numit browser pentru a accesa documente și pagini web
  • navigarea pe alte pagini se face prin intermediul hyperlink-urilor
